Python

Lately we’ve been learning new features via python and they have been been remarkably useful in many aspects including biology. When working in labs, there are times when we need to analyze DNA sequences but it can get quite tedious to do it by hand. A good example of the application of python in biological analytics was in Homework 11 from my SDS348 class. It utilized a sequence of characters known as string that in this case contained DNA nucleotide bases and asked about its fragmentation. The function re.findall() was used to find all the matches and return them as a list of strings, with each string representing one match. It should be mentioned that a neat feature learned in this class was the usage of library(reticulate) in order to employ python in R so that you can use both coding languages alongside each other! Additionally, I’ve attached a picture of some of the most common python codes that were handy in keeping track of the basic codes when working on this homework assignment.

DNA Question

Note that per the IUPAC nucleotide code, N is any base, R is A or G, W is A or T. How many cuts total in the sequence do you expect if you digest with both of these restriction enzymes? How many fragments do you expect?
